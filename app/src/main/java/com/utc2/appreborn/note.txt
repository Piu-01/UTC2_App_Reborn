com.utc2.appreborn
├── data                (Dữ liệu: API, Database, Repository)
│   ├── local           (Room database, SharedPreferences)
│   ├── remote          (Retrofit API interfaces, DTOs)
│   └── repository      (Nơi kết nối giữa Local và Remote)
├── ui                  (Giao diện & Logic điều khiển UI)
│   ├── login           (LoginActivity, LoginViewModel)
│   ├── main            (MainActivity, MainViewModel)
│   ├── components      (Các UI nhỏ tái sử dụng: CustomView, Button, Loading, Animation,...)
│   └── adapter         (Các bộ đổ dữ liệu cho RecyclerView)
├── model               (Các lớp thực thể - Data classes)
├── di                  (Dependency Injection - Hilt/Dagger nếu có)
├── utils               (Các lớp hỗ trợ: Constants, Extensions, Helpers)
└── base                (Các lớp BaseActivity, BaseFragment dùng chung)
